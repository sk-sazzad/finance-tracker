# 💰 Finance Tracker

**A powerful personal finance management app built with modern Android technologies.**  
Track your income, expenses, budgets, and get detailed analytics — all in one place.

[![Download APK](https://img.shields.io/badge/⬇️%20Download%20APK-v1.0.0-brightgreen?style=for-the-badge)](https://github.com/sk-sazzad/finance-tracker/releases/download/v1.0.0/Finance-Tracker-v1.0.0.apk)

</div>

---

## ✨ Features

- 🏠 **Dashboard** — Get a quick overview of your financial health
- 💳 **Transactions** — Add, edit, and manage income & expenses
- 📊 **Analytics** — Visual charts and insights on your spending
- 🎯 **Budget Tracking** — Set budgets and track progress in real-time
- 🛠️ **Tools** — Useful financial calculators and utilities
- 🌙 **Dark Mode** — Fully supports light and dark themes

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| Language | Kotlin |
| UI Framework | Jetpack Compose + Material 3 |
| Architecture | MVVM + Repository Pattern |
| Database | Room Database |
| Navigation | Navigation Compose |
| Networking | Retrofit + OkHttp |
| JSON Parsing | Moshi |
| Preferences | DataStore |
| Build System | Gradle (KTS) |

---

## 📱 Requirements

- Android 7.0 (API 24) or higher
- ~19 MB storage space

---

## 🚀 Installation

### Option 1 — Direct Download
Click the **Download APK** button above and install on your Android device.

> ⚠️ Enable **"Install from unknown sources"** in Settings before installing.

### Option 2 — Build from Source
```bash
git clone https://github.com/sk-sazzad/finance-tracker.git
cd finance-tracker/finance-tracker
./gradlew assembleDebug
```
